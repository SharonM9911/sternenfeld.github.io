Climbing Stairs

Category Difficulty Likes Dislikes
algorithms Easy (51.58%) 14031 414

You are climbing a staircase. It takes n steps to reach the top.

Each time you can either climb 1 or 2 steps. In how many distinct ways can you climb to the top?

Example 1:

Example 2:

Constraints:

  • 1 <= n <= 45

答案：

  1. 循环数组
  • runtime beats 40.11 % of c submissions
  • memory usage beats 18.26 % of c submissions (5.7 MB)
  • 时间复杂度:O(n), 循环n次每次花费常数的时间代价
  • 空间复杂度:O(1), 常数个变量

2. 矩阵快速幂

  • runtime beats 100 % of c submissions(0ms)
  • memory usage beats 45.21 % of c submissions (5.5 MB)
  • 时间复杂度O(log n)
  • 空间复杂度O(1)

分析：

这道题一开始不会做，真的是没有一点思路，然后看了答案知道是动态规划循环数组解法，以及还存在的快速幂解法，总而言之，线性代数真的有用，感谢hhy救我狗命.
