Er ist Stark

（又名托尼斯塔克的机器人帝国）

　　（又又名除了武器钢铁侠还能做什么）

　　时间点：托尼搬离回洛杉矶后

　　托尼发现门口来人时自己刚吃完早餐准备去斯塔克工业。从海平面上升起的太阳给波浪镀了层金箔，轻微的海浪声从打开的窗户中传来，这一切让他觉得很轻松。

　　“老板，门口有访客。”星期五拉住了打算前往车库的托尼的脚步，“是罗曼诺夫女士和罗杰斯先生，应该是关于上个月的物资。”

　　托尼做了个手势示意星期五将车辆开到门口，他扣好一下西装的扣子后走到门廊。

　　联盟的飞机就停在房子旁边，亏的新建的马里布大宅足够大，给战机提供了足够的场地。托尼快速走了过去，将墨镜挂在外套的口袋上。

　　“好久不见，小娜。”他和娜塔莎交换了一个贴面吻，然后转向斯蒂夫，“还有队长。”

　　斯蒂夫已经不在意托尼的称呼问题了，自从那天他们两个错过后这还是第一次再见对方，他打量着托尼，除了身影消瘦之外气色还算不错。“我和娜塔莎来是为了…”

　　“上个月的物资对吧，星期五和我说过了，有什么问题吗？”他笑着，手插在口袋里。为了你……斯蒂夫将没说完的句子吞到了肚子里，娜塔莎看了他一眼便开口道：“我们需要一些机器人来做战后重建工作，玛丽亚基金会的负责人说我们最好亲自和你谈谈。”

　　“于是你们就直接飞到了洛杉矶？你应该知道这种事情直接在视频通话中就能做到的。”

　　“此外我们想看看你过得怎么样，你退出联盟之后大家都很担心你。”斯蒂夫拦住了娜塔莎直接开口，他不想再兜兜转转了。

　　托尼张了张嘴什么都没说，眉眼里的疏离淡了许多。正巧车停在了三人面前。托尼走到主驾驶旁拉开了车门。

　　“上车吧，斯塔克工业会有你们想要的东西的。”

　　——————————

　　任何一个学过普通生物学的人都知道，按照s曲线，生物数量等于K环境容纳量的一半时，增长速度最快。不过人类似乎从未达到自身的K值，所以人数削减一半的下场就是根本无法恢复到原本的社会秩序，但这的确给了动植物一个很好的发展机会，他们不会悲伤，也不会落寞，只是一个劲的拼了命疯长，尝试将自己的生存范围扩展到整个城市。

　　而托尼，就是那个以自身智慧与实力维持这个社会正常发展的人。他只用了一个月将斯塔克工业重整，调整计划，并在接下来的时间内发展所有他能想到的机器人领域，从微纳米级别到正常尺度，再大型交通领域。他和他的一群天才们用机器人填补了空缺的人类数量，以一种人类根本没有察觉却又深受影响的方式。

　　而这一切除了托尼，没人能做得到。

　　本该堵到爆炸的洛杉矶如今出现了畅通无阻的路况，他们从马里布疾驰，一路上所有人都没有说什么，车里保持着诡异的安静，只有车载音响传出几句德语。

　　黑色跑车停在了车库里，托尼拉开车门带着两人进入电梯。“星期五，今天的日程安排？”

　　“老板，您在9点整需要参加董事会，预计10点结束。下午2点半与清洁能源的项目负责人有一场报告，今天的研发计划一共3项，预计在今晚8点可以完成任务。”

　　“好的。昨天做报告的那个负责人叫什么来着？卡瑞…”托尼想了一下，却还是没想起来自家机器人部门的负责人名字。

　　“卡洛斯·梅亚，老板，他在今天上午10点前都没有安排，目前已经到达实验室，需要我安排他为罗曼诺夫女士和罗杰斯先生介绍公司最新的研发成果吗？” 　　

　　“那小娜，罗杰斯，我们先去见见卡洛斯吧。”

　　电梯在研发部的楼层停了下来，斯蒂夫没有理会娜塔莎的眼光，温柔的注视着托尼的侧脸，不到半年的时光，他终于恢复了正常状态，甚至比离开基地前还要健康。

　　“欢迎来到斯塔克的奇妙世界——”

　　当一行人在踏出电梯时，一个与外界破败完全不同的世界出现在了眼前，娜塔莎被漫天飞舞的光屏小小震惊了一下。即使还不到正常上班时间，好几个研发团队已经在工作了，咖啡机一刻不停的工作着，运输机器人则负责把这些新鲜出炉的饮料送到每个工程师的手上。仔细看去，诺大的研发部门用透明的隔音材料分成了一个又一个小块，每个研发团队资料被飞舞的机械手摆放的整整齐齐，运输各种物资的机器人井然有序，像极了拥有良好训练的军队。

　　“托尼，你真的是太……”

　　“老板，普朗克已经为您通知了梅亚先生，他会在3分钟后到达，以及，波兹小姐已经达到，她让我转告您，会议上会商议重建机器人的发售。”

　　星期五像是护主的小狼一样没有来的对斯蒂夫充满敌意，她毫不留情的打断了他的话，为托尼汇报着最新的情况。

　　托尼宠溺的笑了一下，轻轻拍了下队长的手臂后又快速抽离，插在口袋里带着两人拐了个弯，正巧遇到前来的卡洛斯。

　　斯蒂夫看着前来的金发碧眼亲密的搂了一下托尼，他的心脏像是被人狠狠捏了一下。他压抑着自己的情绪，眼睛里却涌动着不满。娜塔莎察觉到了气氛的变化，按住了斯蒂夫的拳头，调整了自己的表情。

　　“卡洛，项目怎么样了？”托尼拍了拍自己提拔上来的项目负责人，咖啡色的大眼睛里带着笑意，“进展特别顺利，预计明天开会的时候第一代就能完成了。上次你提的建议特别有用，天呐，你真不愧是斯塔克！(Du bist Stark!)”卡瑞说着，脸上对托尼的敬佩简直是喷薄而出。“对了，今天我能为您做些什么，我尊敬的斯塔克先生？(Mein lieber Herr Stark?)”

　　190的男人在托尼面前像是个讨要糖果的孩子，他的面庞带着德国人特有的气质，出口的德语带了些俏皮的气息。而这一切在斯蒂夫眼里充满了不可言喻的意味，即使到21世纪已经有五六年，那些留在上个世纪里对纳粹的恨意又蔓延而上。他像极了为了争夺喜欢的人的注意力的毛头小子，急切的挤到托尼身边争夺关注。

　　托尼察觉到了斯蒂夫的不对劲，他扭过头扫了斯蒂夫一眼，无奈地叹了口气：“卡洛，这是斯蒂夫·罗杰斯aka美国队长，和娜塔莎·罗曼诺夫女士，一会儿你就带着他们看看我们最新的项目，尽量满足他们的需求……”托尼向梅亚介绍两人，娜塔莎轻点下巴示意，而斯蒂夫的目光似乎要把伍德大卸八块。“我得去开会了，星期五，将普朗克的基础权限向罗杰斯和小娜开放。”托尼看了看手表，下意识拍了拍斯蒂夫的手臂，然后快步走向了电梯。

　　直到电梯门关上，托尼像是脱力一般靠着墙大口喘着气。只有他自己知道在罗杰斯在的范围里他的心跳跳的有多快，像是一脚踩下油门的赛车，不管不顾的跳跃着。甚至说他的大脑都在违背自己的意志不停释放着那些让他感到快乐的激素，他太开心了，浑身上下的细胞都在唱歌跳舞，而他完全无法抑制这些。

　　无论发生什么，他都深爱着斯蒂夫，托尼永远不会怀疑这个，就像他永远不会怀疑自己创造的一切。

　　一脸宠溺的看着托尼离开的斯蒂夫转过身面对卡洛时就变了个脸。梅亚硬生生从他面无表情的脸上看出了不满，他挠了挠一头金毛，值得尴尬的笑了笑。

　　“普朗克？”娜塔莎出口缓和了两人之间的气氛。

　　“您好，罗曼诺夫女士，我是斯塔克工业设计并开发的人工智能助手，主要工作是帮助研发部门的工程师设计与创造。您有任何需求都可以召唤我。”带了些德语口音的男声回荡在三人身边，梅亚变得兴奋起来，他手舞足蹈的向二人介绍关于普朗克的事情：“普朗克是物理史上和爱因斯坦先生并列的人，他是量子力学的开创人之一，是物理学乃至世界上里程碑一样的人物。而普朗克是我加入斯塔克工业后和斯塔克先生一起参与的第一个项目，我们一致想要让普朗克成为这个世界划时代的项目，几乎所有的斯塔克工业机器人都有普朗克的参与，可以说只要有斯塔克工业的机器人在，这个世界就会有普朗克存在。”

　　他们走着，绕过了一个又一个透明格子，准备走向员工电梯。一路上每个格子逐渐被穿着白大褂的工程师填满，他们忙碌着，只为拯救这个世界。

　　罗杰斯突然意识到，托尼的军队并非只是他的钢铁军团。当托尼离开的时候，他内疚过，绝望过，却从来没有想过，托尼斯塔克为什么要离开的原因。

　　不仅仅是他的身体没办法承受战甲了，托尼斯塔克之所以是托尼斯塔克并不是因为铠甲，早在他成为钢铁侠时他就已经是个英雄了。

　　或者说坐拥庞大斯塔克工业帝国的国王？他一定会爱死这个称呼的，斯蒂夫想着，不自觉勾起了嘴角。

　　“每个斯塔克机器人都有普朗克吗？”他出声问道，像是在确认什么。

　　“上帝啊。”卡洛斯嘟囔了几句德语，吐槽着队长，“你又在担心奥创了吗？”

　　“不，不是……”被戳破心事的队长瞬间红了脸，他就是忍不住担心这个。不是说他不信任托尼，只是他不想再看到托尼失去贾维斯的悲伤了，前提是普朗克有和贾维斯一样的地位的话。

　　“不会发生和奥创一样的事了，队长，请您不要担心这个，斯塔克先生有着你们永远无法想象的天赋，他不仅仅创造，而且革新。”

　　卡洛按下电梯按钮，向娜塔莎和斯蒂夫解释。

　　“所以说，普朗克参与创造了那些机器人，他设计了一部分结构，电气或者其他之类的。”

　　卡洛被娜塔莎的回答惊讶到了，他蓝色的眼睛里瞬间灌满了惊喜。“对，就是这样，先生从未尝试让机器真正的思考，他只是试着让机器在原有数据的基础上发展。而奥创不同，他并不是机器进化而来的，他本身就有生命。不过可惜了贾维斯，本来他也有机会重生的。”

　　“贾维斯难道不是被奥创摧毁了吗？”斯蒂夫觉得自己的后背发凉，不好的预感攀着脊柱涌上大脑。卡洛看了看娜塔莎，又看了看队长，良久才发出一声冷笑：“不是的，队长。奥创确实攻击了贾维斯，但是先生将他重新带回了。我当时跟着先生处理整件事情，带领整个开发团队解析数据流。先生把贾维斯带回时他真的很开心，甚至拒绝了团队提出的一起修复的建议，他想要自己修好贾维斯。”

　　“他当时想要激活再生摇篮……”那感觉就像是被闪电狠狠劈在头上——他从来都没有相信过托尼。

　　幻视出生时娜塔莎并不在大厦里，她了解到的事实也存在欠缺的地方，而唯一知道最完整事实的布鲁斯却在战斗结束后消失了。然后就是三巨头分开，娜塔莎很少在基地再见到托尼，直到法案出现。她看了看队长，又看了看卡洛，胸腔内涌上了让她没法透气的内疚感。他们没有任何一个人，任何一个，想到托尼失去了什么，却都在法案产生时不约而同的责怪他。上帝啊，他们到底是一群怎样的队友啊！

　　电梯门缓缓打开，三人踏入封闭的空间里，却怀着不一样的心情。

　　————————

　　斯塔克工业总部并不像曼哈顿的斯塔克大厦一样将所有部门放在一起，他反而是扩大了占地面积，层层叠叠的四五层楼堆叠形成了一个工业园区，每层楼之间都是快速通行的连廊，如果特别急的话还有水平移动的电梯在楼宇之间来回运行。而娜塔莎一行人踏上的就是通往产品部门的电梯，从透明连廊看去就是园区中心的方舟反应堆。不时有光芒流动，彰显着园区主人无与伦比的天赋。

　　“园区的方舟反应堆原本只是为园区供电，大战后洛杉矶的电力供应不上，整个城市供电时断时续，根本无法维持正常运行。斯塔克先生以成本价与政府签下订单，利用我们研发的建筑机器人改造电路使方舟反应堆的输入电力开维持整个市的电力供应。原先的发电站工人经过培训后并入方舟反应堆的维护队伍中，到目前为止，整个城市的电力开始恢复到大战前的正常水平。”

　　卡洛斯看着两人，开口解释道。眼中是满满的倾佩之情。他知道斯塔克先生不仅仅是个商人，他一直都是一个英雄，无论是否有那件铠甲。

　　“这份订单不仅仅把斯塔克的清洁能源推向了世界顶端，同时也是在告诉世界，斯塔克的机器人完全有能力提供原本只能由人力提供的服务。而这些正是所有国家都需要的东西。我们不能停在原地自怨自艾，走的人已经走了，留下的人还要继续生活，无论大家的选择是什么，斯塔克工业确保大家有选择的权利。”

　　“现在斯塔克工业和多少国家签订了合同？”斯蒂夫突然开口问道，机器替代人力的方案确实值得推荐，但一旦处理不好就可能引起矛盾和纠纷。

　　“按照目前市场上的需求来看，只有斯塔克一家能提供有足够输出功率的清洁能源，即使像是北欧一些主要使用风能，水能，太阳能的国家也基本有五六个订单。”娜塔莎抱着手臂，她明白队长在想什么，“但是托尼最厉害的一点是，每一个政府想要采购斯塔克企业的清洁能源与配套产品，就必须让原本就在相应岗位的工人参与进来，然后斯塔克工业会提供机器人来参与剩下的工作。”

　　成本价，强制要求工人参与，托尼根本就不是在进行商业，他完全就是在救灾。斯蒂夫握紧了拳头却又不得不放开，这是一场属于托尼自己的战争，他帮不上任何忙。

　　电梯到了产品部门，他们停了下来，卡洛斯带着两人往特种机器人方向走去，边走边向两人解释。

　　“罗曼诺夫小姐说的没错，斯塔克工业从08年关闭武器部门后将大量资金转向了机器人研发部门，并首先研发了十种民用产品。到16年时除了最开始的十种产品的迭代与研发团队，机器人部门又新增了67个研发团队157种产品，从普通的家用到航空航天部门，超过63%的领域都有斯塔克工业机器人的身影。”

　　“12年外星入侵战争，战后的收缴的一些外星武器为研发部的供能与结构提出了新的灵感，之后我们的产品适用性更加广泛。除了刚才提到的157种旗舰产品，我们还有几百种实验性产品，主要用来测试研发团队的想法与设计。而这场战后，先生将所有的实验性产品全部推向市场使用，而实践证明，先生永远都不会错。到目前为止，已经有超过2/3的国家使用斯塔克工业的产品来做灾后重建，同时大量的产品制造也让我们再次盘活了整条产品制造链。”

　　相较于研发部门，产品部门的架构则更倾向于介绍，每个区域的装饰都与产品的设计有着或多或少的共鸣。斯蒂夫站在民用设备展示厅的中央，卡洛斯为娜塔莎介绍着产品，他环视着整体的设计风格，看起来般配的设计里总让他觉得缺少了什么。

　　娜塔莎和卡洛斯的声音逐渐减小，他们的身影逐渐消失。斯蒂夫快步跟上他们的脚步，直到一个更加光怪陆离的世界出现在面前。

when i = 1, a[1]=1, b[1]=1, carry=3, then ans[1]=’1’, and carry=’1’now,

when i=2, a[2]=1, b[2]=0. Need to be mentioned that here, because the length of b is shorter than 2, so we need to fill in with 0 at the bit. Then carry=2, and ans[2]= ‘0’, carry became ‘1’ again.

when i =3, a[3]=0,b[3]=0, carry=1,ans [3]=1,carry became 0;

when i = 4, a[4]=1, b[4]=0, ans[4]=1, carry=0;

当计算结束，将结果反转后加**’\0’的终止符**后输出即可
